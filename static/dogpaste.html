Dogpaste uses 11-character codes (6-char encryption key + 5-char ID):

  • Human-friendly: Uses charset 23456789abcdefghjkmnpqrstuvwxyz (no ambiguous chars like 0/O, 1/l/I)
  • Guessable: 29^6 = ~594 million key combinations (not cryptographically secure)
  • PBKDF2: Keys are derived using 100,000 iterations to slow down brute force
  • No forward secrecy: Anyone who guesses the code can decrypt your paste
  • Short URLs: Super easy to type and share but less secure than full-length keys
